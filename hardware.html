HARDWARE

El hardware se refiere a los componentes físicos de un sistema informático que puedes ver y tocar. Esto incluye dispositivos como la unidad central de procesamiento (CPU), módulos de memoria, discos duros, teclados, mouse, monitores y otros periféricos. Para ver mas imágenes, hacer clic

Tipos de Harware

    Segun su función
  • De procesamiento: Es el corazón del sistema, responsable de ejecutar las instrucciones y realizar cálculos. Ejemplos: CPU, GPU, placa base, memoria RAM
  • Hardware de almacenamiento: Almacena datos y programas. Puede ser primario (interno, como la RAM) o secundario (externo, como discos duros o memorias USB).
  • Hardware de entrada: Permite al usuario introducir datos o comandos al sistema. Ejemplos: teclado, ratón, micrófono, escáne
  • Hardware de salida: Muestra o entrega información al usuario. Ejemplos: monitor, impresora, altavoces.
  • Hardware de entrada/salida (E/S): Realiza ambas funciones, entrada y salida de datos. Ejemplos: pantalla táctil, unidad flash USB, disco duro externo
    • Según su ubicación
  • Hardware interno: Componentes integrados dentro del dispositivo principal, como la placa base, la CPU, la RAM, etc.
  • Hardware externo: Periféricos conectados al dispositivo, como teclado, ratón, monitor, impresora
  • Hardware híbrido: Elementos que pueden ser internos o externos, o funcionar de ambas formas, como una cámara web integrada o un disco duro externo.
